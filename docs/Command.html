    Command(options)

    A command.

    new Command(options)

    Parameters:
    Name Type Description
    options CommandOptions The command options.
    Properties:
    Name Type Attributes Description
    args Array.<Argument> The command's arguments.
    botPermissions Array.<string> The permissions required by the bot to execute the command.
    cooldowns Cooldown <nullable>
    		 An object of all user cooldowns and cooldown options on the command.
    description string The command's description.
    group Group The command's group.
    hasCooldown boolean Whether or not the command has a cooldown.
    memberPermissions Array.<string> The permissions required by the invoker to use the command.
    names Array.<string> The command's names.
    preconditionOptions Array.<*> The options to be passed to preconditions when they're run.
    preconditions Array.<Precondition> The preconditions to be ran on the command.
    postconditionOptions Array.<*> The options to be passed to postconditions when they're run.
    postconditions Array.<Postcondition> The postconditions to be ran on the command.
    usableContexts Array.<Symbol> An array of contexts the command can be used in.
